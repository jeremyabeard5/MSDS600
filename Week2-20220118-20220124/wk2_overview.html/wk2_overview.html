Week 2: Cleaning and Preparing Data

Overview

This week starts by describing different types of data, including qualitative and different types of quantitative data.  Characteristics of data describe its organization or attributes, such as time-series data or the dimensionality of the data.  Although the specific data processing details will differ depending on the type of data and its characteristics, the processing steps will be much the same – data acquisition, data extraction, data preprocessing, data analysis, and data presentation - essentially the data science process we talked about last week.

We will also learn about data preparation and cleaning. This is an important step in a data science process, especially if we are using machine learning. The way we clean, preprocess, and feature engineer data has a significant impact on the final results of analysis or machine learning systems. It's often said that data scientists spend most of their time cleaning and preparing data (anywhere from 75-90%), so it is an important skill to learn and master.

Learner Outcomes

Upon the completion of this week, you should be able to

  1. Describe the different types of qualitative and quantitative data.
  2. Explain the characteristics of data sets.
  3. Discuss the cleaning and preparation stage of the data science process.
  4. Carry out basic data cleaning and preparation in Python.
  5. Improve your understanding and skills for using Jupyter Notebooks, Anaconda, the command line, and Python package managers.
